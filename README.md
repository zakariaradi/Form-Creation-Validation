# Form-Creation-Validation

A simple and practical project focused on HTML forms, JavaScript validation, and DOM manipulation.

# Overview

This project implements a user registration form with full client-side validation using JavaScript.
The goal is to help learners understand how to:

Validate form inputs

Interact dynamically with DOM elements

Provide useful user feedback

Prevent invalid data submissions

# Learning Objectives
✔ 1. Form Validation

You will learn how to validate:

Username length (minimum 3 characters)

Email format (must contain @ and .)

Password length (minimum 8 characters)

✔ 2. DOM Manipulation

Selecting elements using getElementById

Updating text dynamically

Displaying styled messages for validation

✔ 3. Event Handling

Handling the submit event

Preventing default form submission

Showing success/error messages interactively

# Validation Rules
Username

Must be at least 3 characters

# Email

Must include both: @ and .

Password

Minimum 8 characters

# User Feedback

If all validations pass:

A green message appears:
"Registration successful!"

If validations fail:

Red error messages appear, such as:

"Username must be at least 3 characters long."

"Please enter a valid email address."

"Password must be at least 8 characters long."
